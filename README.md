# Cipher Toolkit

A command-line encryption tool supporting 3 classical ciphers.

## Ciphers
- **Caesar** — shifts letters by a numeric key
- **Vigenère** — shifts letters using a keyword
- **XOR** — encrypts using bitwise XOR operation

## Usage
```bash
python cipher_toolkit.py
```
